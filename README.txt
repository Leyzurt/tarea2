Proyecto extremo

Instrucciones de uso:
	1 - Cámara: para mover la cámara hacia delante, izquierda, abajo y derecha usar las teclas: W A S D respectivamente. Para moverse hacia arriba y hacia abajo usar C Z respectivamente. Para mover el target (hacia donde mirar) usar el click DERECHO del mouse.
	2 - Raycasting: implementado para 3 objetos en 8 bits Head, Bonsai y Skull. Estos pueden seleccionarse mediante el combo box y hacer clic en "Select". Con el objeto seleccionado se puede hacer manejo de todas las transformaciones.
	3 - Función de transferencia: implementado para todos los objetos en la escena, los puntos de control se pueden crear haciendo clic en las coordenadas deseadas, para seleccionar un punto de control basta con hacer clic en el y arrastrarlo para moverlo, una vez seleccionado se puede eliminar mediante la interfaz además de cambiar el color (mediante un picker en la misma).
	4 - Varios volúmenes al mismo tiempo implementado.
