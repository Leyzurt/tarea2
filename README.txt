Instrucciones de uso:
	1 - Cámara: para mover la cámara hacia delante, izquierda, abajo y derecha usar las teclas: W A S D respectivamente. Para moverse hacia arriba y hacia abajo usar C Z respectivamente. Para mover el target (hacia donde mirar) usar el click DERECHO del mouse.
	2 - Modelo principal: el modelo principal está dado por un "dragón" el cual se puede rotar, trasladar, y cambiar algunas propiedades.
	3 - Luz/Shader: Se usa una luz representada por una linterna, en la interfaz se pueden cambiar los parámetros de luz, dirección, color, etc. Igualmente para el shader. NOTA: los shaders/paramatros del shader, se cambiarán todos los modelos en la escena.
	4 - Animaciones: Para detener o activar las animaciones hacer clic en "Animaciones" en la interfaz.
	5 - Cube map dinámicos: implementado refract esfera / reflect elemental. NOTA: se puede activar o desactivar el refract/reflect (así como su refract ratio) para cualquier objeto de la escena.
	6 - Normal mapping: implementado para los objetos dragón - arma de fuego.
	7 - Sombras: las sombras se pueden cambiar de planares a shadow mapping cambiando el tipo de luz (point light para planares, reflector/direccional para shadow mapping). NOTA: la dirección de la luz se puede cambiar en la interfaz.
